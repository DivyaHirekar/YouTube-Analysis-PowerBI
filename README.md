# 📊 YouTube Performance Analysis Dashboard

## 🧾 Objective
This project analyzes YouTube video performance data to understand trends in views, likes, comments, and audience engagement. The goal is to identify high-performing content categories and key patterns over time.

---

## 📂 Dataset
- videos-stats.csv  
- comments.csv  

The dataset contains information about video metrics such as views, likes, comments, and keywords.

---
## 🛠️ Approach

### 1. Data Cleaning
- Removed duplicate records  
- Handled missing values  
- Standardized data formats  

### 2. Data Processing
- Created calculated columns  
- Structured data for analysis  
- Built relationships between datasets  

### 3. Data Analysis
- Analyzed views by keyword  
- Compared engagement (views vs likes)  
- Studied trends over time  

### 4. Dashboard Creation
- Built interactive dashboard using Power BI  
- Used KPIs, charts, and filters for insights  

---

## 📈 Key Insights
- Total Views: 22 Billion+  
- Total Likes: 316 Million+  
- Total Comments: 15 Million+  
- Keywords like **Google** and **Animals** generated highest views  
- Strong relationship between views and likes
